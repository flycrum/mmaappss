# getting-started plugin

Skills for first-time setup, local testing, and ongoing maintenance of `@mmaappss/sync` in consumer projects.

## Purpose

- **setup-local-testing** — Local link or pack; wizard; wire-up target; run sync; verify
- **setup-from-npm** — After npm install; wizard; wire-up; verify
- **doctor** — Diagnose sync failures, config issues, version/schema mismatches
- **updates** — Refresh sample plugins and templates in target
- **remove-mmaappss-sync** — Teardown wizard for local or npm install

See **AGENTS.md** for agent-facing summary and skill usage. See parent [getting-started README](../../README.md) for bundle overview and mermaid flow.
